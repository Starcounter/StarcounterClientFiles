Collection of UI components that are used as wrappers over native elements. Custom elements that work well in light DOM as well as in shadow DOM to render custom UI controls based on data from native elements distributed from a different tree.

Table of Contents

  • Principles & solution overview
  • <uni-data-table> - a data table with lazy loading and rich declarative cell content
  • <uni-date-picker> - date picker UI for native <input type="date">
  • <uni-form-item> - form item container for a <label>, a form control of any kind, and an <output> validation message
  • <uni-form-item-group> - group container for <uni-form-item> elements, with its own <label>, and an <output> validation message
  • <uni-pagination> - easy pagination element for native <input type="range">
  • <uni-popover> - A simple popover with automated popover arrow to your target element.
